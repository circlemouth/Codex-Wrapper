# Codex Wrapper Agent 指示サンプル

このファイルは `workspace/codex_profile/codex_agents.md` を作成する際の雛形です。
`codex_agents.md` にリネーム（もしくは内容をコピー）すると、サーバー起動時に Codex の `AGENTS.md` に反映されます。

## 設計方針例
- プロジェクト固有のコーディング規約
- テストやドキュメント更新の必須手順
- セキュリティに関する禁止事項や承認フロー

## テンプレート
```
# プロジェクト方針
- ...

# 禁止事項
- ...

# 参考資料
- ...
```

必要なセクションだけを残し、プロジェクト用に編集してください。
